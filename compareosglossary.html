Glossary for Compare OS
Other Windows 10 MacOS
Pre-Built - A computer that was pre built before you got it Microsoft = The company that made Windows 10 Apple - The company that made macOS
Custom Built - A computer that you assembled on your own with all the parts Internet Explorer - Windows 10 built in web browser Safari - macOS built in web browser
CMD - Windows 10 program to run commands and start programs with, most programs access this by default in the background Terminal - macOS program to run commands and start programs with, most programs access this by default in the background
PowerShell - The less used newer and more powerful version of CMD on Windows 10 Hackintosh - a Custom Built pc that runs macOS without approval of apple regularly to run macOS better
